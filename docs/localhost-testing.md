# Localhost Testing

## Cara Menjalankan

1. Buka `dreamlab-site/start-localhost.cmd`
2. Tunggu sampai terminal menampilkan `Ready`
3. Buka:
   - `http://127.0.0.1:3000/thankyou/google`
   - `http://127.0.0.1:3000/thankyou-medsos`
   - `http://127.0.0.1:3000/ads/thankyou/google`

## Catatan

- Jangan tutup window terminal selama testing.
- Jika `localhost` bermasalah di browser, pakai `127.0.0.1`.
- Jika port 3000 sedang dipakai, hentikan proses lama dulu lalu jalankan ulang.
